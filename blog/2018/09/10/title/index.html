這是我重新開始寫 Markdown，希望能堅持下去 ^_^

環境：Ruby 2.4.3、Rails 5.2.1、Ubuntu 16.04.5LTS、Vim 7.x

目前會的Gem：

  • bootstrap 4 (HTML框架)
  • font-awesome-rails (網頁向量字體)
  • mysql2 (資料庫)
  • bootstrap_form (Bootstrap表單)
  • devise (帳密驗證)
  • cancancan (權限分級)
  • carrierwave (上傳圖片)
  • mini_magick (裁剪圖片)
  • kaminari (分頁)
  • jquery-rails (jQuery)
  • fullcalendar-rails (月曆)
  • momentjs-rails (MomentJS)
  • pry (很炫的Console)
  • byebug (除錯用)
  • better_errors (除錯用)
  • ransack (欄位即時排序)
  • passenger (網站佈署)

RoR會的技能：

  • controller (CRUD、表單欄位驗證、Layout、before_action、CRSF跨站請求偽造)
  • model (1對1、1對多、多對多、驗證、回呼、排序、自定方法)
  • view (迴圈、迭代、render partial、表單[simple_form_for、bootstrap_form_for]、元件[input、select、checkbox、textarea])
  • assets (SASS、jQuery)
  • helper (自定輔助方法)
  • routes (resources、root、nest、namespace、shallow、member、collection )
  • scaffold、migration、destroy
  • rails 5.2.1 new 專案 -d mysql
  • rails g scaffold 名稱 欄位:屬性
  • rake db:migrate / db:rollback / db:seed / db:setup
  • deploy with capistrano / passenger
